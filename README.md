# Chess AI - Human vs AI

This project is a browser-based chess application developed using HTML, CSS, and JavaScript. It allows a human player to compete against an AI opponent that uses the Minimax algorithm with Alpha-Beta pruning for move selection.

The application is fully self-contained within a single HTML file, making it lightweight, easy to deploy, and simple to maintain.

## Features

- Complete chess rules implementation
- AI opponent using Minimax search
- Alpha-Beta pruning for optimized move calculation
- Multiple difficulty levels
- Move history tracking
- Undo last move functionality
- Captured pieces display
- Pawn promotion support
- Castling support
- Check, checkmate, and stalemate detection
- Clean and responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Minimax Algorithm
- Alpha-Beta Pruning
- Piece-Square Table Evaluation

## How to Run

1. Download or clone the repository
2. Open the `index.html` file in any modern web browser

No additional setup or installation is required.

## Deployment

This project can be deployed easily on:

- Netlify
- Vercel
- GitHub Pages

For the fastest deployment:
- Rename the file to `index.html` if needed
- Upload it directly to Netlify Drop or Vercel

## AI Implementation

The AI opponent is based on:

- Minimax algorithm for exploring future board states
- Alpha-Beta pruning for reducing unnecessary calculations
- Piece-square tables for positional evaluation
- Adjustable search depth for difficulty control

This provides a strategic and efficient chess-playing experience directly in the browser.
